- 減重計畫規劃

## 組員

## 專案簡介
- 本專案提供使用者輸入個人身體資料、設定每週運動目標、紀錄每日運動內容，並即時計算本週目標達成度。

## 使用技術
- **HTML** 
- **CSS**
- **Bootstrap**
- **JavaScript**
- **Constraint Validation API**

## 功能特色

### 1.個人資料管理
- 使用者輸入姓名、性別、年齡、身高、體重、目標體重、目標運動量
- 計算BMI與體重目標差距
- 可刪除個人資料並重新設定

### 2.運動紀錄功能
- 輸入運動種類、日期、強度、時間、備註
- 自動計算本次運動估計消耗熱量（MET × 體重 × 時間）
- 非本週的日期禁止加入（並跳出提醒）
- 使用卡片動態呈現在畫面上
- 可刪除任一筆運動紀錄

### 3.本週目標進度條
- 使用者可設定每週運動目標（分鐘）
- 自動統計本週累積運動時間
- 以動態進度條呈現
- 限制只計算本週一～本週日的紀錄

### 4.深色模式切換
- 一鍵切換深色/淺色模式

### 5.RWD響應式網頁
- 各種裝置皆可正常使用  
- Header、卡片區會在窄螢幕自動調整

### 6.資料儲存
- 個人資料、運動紀錄、深色模式設定都存於localStorage
